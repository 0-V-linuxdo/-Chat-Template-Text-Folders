# [Chat] Template Text Folders
简介：
在AI网页底部，添加 `提示词模版` 文件夹+按钮，
减少重复输入，提升输入效率！

提示：
脚本数据存储在 本地 - 浏览器localstorage 中！

---

支持的浏览器：
- Chrome
- Firefox
- Safari

注意：
Firefox/Safari 获取剪切板内容，需要手动确认！

---

支持的脚本管理器：
- Adguard (app version) - All
- violentmonkey - Chrome/Firefox
- Stay - Safari
